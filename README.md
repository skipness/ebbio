# ebbio
An unofficial Flutter showcase app of [ebb.io](https://ebb.io)

## Getting Started
1. Install [Flutter](https://flutter.io/setup/)
2. Clone this repo
3. Move to project directory
4. Run `flutter packages get`
5. Run `flutter run`
